# FIFA 2023 Dataset

Este projeto apresenta um conjunto de dados do FIFA 2023, onde a seleção de times e jogadores exibe suas estatísticas. O projeto foi desenvolvido usando Streamlit e Python.

## Origem dos Dados

Os dados foram obtidos do site Kaggle, do conjunto de dados [FIFA Player Stats Database](https://www.kaggle.com/datasets/bryanb/fifa-player-stats-database).

## Como Executar o Projeto

1. Clone o repositório para a sua máquina local usando `https://github.com/anselmoaxo/dataset-fifa23.git`
2. Navegue até o diretório do projeto.
3. Instale as dependências necessárias com `pip install -r requirements.txt`
4. Execute o aplicativo Streamlit com `streamlit run app.py`

## Funcionalidades

Ao selecionar um time e um jogador, o aplicativo exibirá as estatísticas relevantes do jogador.
